Disable Microsoft Application Experience Program

ABOUT
Remove Telemetry reporting for the Microsoft Application Experience Program
Program (MS AEP)

	
CONFIGURATION
The following options can be added to config.ini

	DISABLEMSAEP (Boolean) - Enable or disable MS AEP
		Y	- Enable (DEFAULT).
		N	- Disable.


NOTE
Disabling the plugin does not undo changes made in previous runs.
